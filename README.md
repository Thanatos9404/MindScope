## 🚀 Quick Start

### **Option 1: Use the Live Application**
Simply visit [http://mindscope-yt.vercel.app](http://mindscope-yt.vercel.app) and start your mental wellness assessment immediately.

### **Option 2: Local Development Setup**

#### **Prerequisites**
- Python 3.8+ installed
- Git installed
- Modern web browser

#### **Clone and Setup**
Clone the repository
```
git clone https://github.com/Thanatos9404/MindScope.git
cd MindScope
```

Backend setup
```
cd backend
python -m venv venv
source venv/bin/activate # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```
Start the backend server
```
python app.py
```

#### **Frontend Setup**
In a new terminal, navigate to frontend
```
cd frontend
```
Start local development server
```
python -m http.server 8000
```
Or use any static file server

#### **Access the Application**
- Frontend: http://localhost:8000
- Backend API: http://localhost:5000
- API Documentation: http://localhost:5000/api/health

## 🎯 Assessment Domains

### **Mental Health Categories**
1. **Mood & Energy**: Depression indicators, interest levels, energy patterns
2. **Anxiety Management**: Worry patterns, nervousness, restlessness
3. **Stress Response**: Coping mechanisms, overwhelm indicators
4. **Sleep Quality**: Sleep patterns, rest quality, sleep disturbances
5. **Overall Wellbeing**: Life satisfaction, emotional balance, social connections

### **Evidence-Based Questionnaires**
- **PHQ-9**: Patient Health Questionnaire for depression screening
- **GAD-7**: Generalized Anxiety Disorder assessment
- **DASS-21**: Depression, Anxiety, and Stress Scales
- **WHO-5**: World Health Organization wellbeing index
- **Custom Sleep Assessment**: Sleep quality and patterns evaluation

## 📝 API Documentation

### **Core Endpoints**
```
GET /api/health # Health check
GET /api/questions # Get assessment questions
POST /api/assess # Submit assessment responses
POST /api/save-user-data # Save user demographics
POST /api/send-email # Email results to user
POST /api/share # Create shareable result links
```
